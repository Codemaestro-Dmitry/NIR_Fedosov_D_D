## Описание
Работа реализует визуализацию поиска наикратчайшего маршрута с помощью алгоритмов _Дейкстры_ и _A*_. Так же в программе присутствует возможность создания пользовательских графов (например, по фотографии схемы магазина) с подложкой и удобного экспорта данных в JSON и PNG.

### Возможности
- Загрузка и сохранение графа в формате JSON;
- добавление подложки к графу;
- построение и отображение пользовательского графа;
- поиск кратчайшего пути между выбранными вершинами (алгоритмы _Дейкстры_ и _A*_ );
- подсветка найденного маршрута на визуализации;
- экспорт результатов в формате PNG.

## Формат входного файла (JSON)
Граф задаётся массивом объектов — каждой вершине соответствует объект со следующими полями:

| Поле      | Тип        | Описание |
|-----------|------------|----------|
| `index`   | integer    | Уникальный идентификатор вершины (нумерация обычно с 1). |
| `x`       | number     | Координата X (float) — для визуализации. |
| `y`       | number     | Координата Y (float) — для визуализации. |
| `z`       | number     | Координата Z (float), можно оставить 0.0. |
| `adj`     | integer[]  | Список индексов смежных вершин. |
| `weights` | number[]   | Веса рёбер (соответствуют порядку в `adj`). |

**Важно:**
- Длина массивов `adj` и `weights` должна совпадать  
- Все веса рёбер должны быть неотрицательными

## Как запустить
На репозитории есть установочный файле "NIR_FedosovDD_setup.exe", с его помощью можно установить программу на свой ПК. Руководство пользователя можно найти в репозитории в файле "Руководство пользователя" или непосредственно в программе во вкладке "Помощь" в меню инструментов.
